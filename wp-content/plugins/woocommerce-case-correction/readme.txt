== Description ==

Customers can sometimes enter their name all in lower case or in other unexpected case. WooCommerce Customer Name & Address Case Correction extension enables you to correct this when they submit their details.

== Usage ==

Simply upload the plugin and you're good to go. There are no options to configure - the extension just works.

== Installation ==

1. Upload the entire 'woocommerce-case-correction' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
